# 😼🐭 Gato vs Ratón

Un juego de consola hecho en **Python** donde cada turno puede cambiarlo todo.

En este proyecto eliges si quieres ser el **gato** o el **ratón** y juegas en un tablero, intentando capturar o escapar de tu oponente. Lo más interesante es que el personaje controlado por la computadora toma decisiones usando **Minimax**, una técnica clásica de inteligencia artificial para juegos.

## ✨ ¿De qué trata?

La dinámica es simple de entender, pero bastante entretenida de jugar:

- Si eres el **gato**, tu objetivo es atrapar al ratón.
- Si eres el **ratón**, debes sobrevivir y escapar durante **20 turnos**.
- Ambos personajes pueden moverse en **8 direcciones**.
- El tablero se representa de forma visual con emojis para que la partida sea más clara y divertida.

## 🧠 Lo más llamativo del proyecto

- Selección de personaje: puedes jugar como gato o como ratón.
- Tablero interactivo en consola.
- IA con **algoritmo Minimax** para decidir movimientos.
- Uso de **distancia Manhattan** como heurística.
- Validación de entradas del usuario.
- Sistema de turnos y condición de victoria/derrota.
- Posibilidad de volver a jugar sin reiniciar el programa.

## 🛠️ Tecnologías y conceptos aplicados

- **Python**
- Programación orientada a objetos
- Listas bidimensionales para representar el tablero
- Algoritmos de búsqueda para juegos
- Validación de datos ingresados por el usuario
- Lógica de control de flujo y turnos

## 🎮 Cómo ejecutar

1. Asegúrate de tener **Python 3** instalado.
2. Descarga o clona este repositorio.
3. Ejecuta el archivo principal con:

```bash
python minimax_lab.py
```


## 📌 ¿Qué demuestra este proyecto?

Este proyecto no solo muestra lógica de juego en consola, sino también una base interesante de **IA aplicada**, modelado de reglas, estructuras de datos y diseño de interacción en terminal. Es una propuesta pequeña, pero con bastante personalidad técnica.
